1. General File and Folder Naming
    - Folders:                 lowercase-with-hyphens
    - JavaScript files:         camelCase.js
    - HTML files:               lowercase-with-hyphens.html 
    - Arduino files:            camelCase.ino
    - Media files:              lowercase-with-hyphens


2. JavaScript Naming Conventions
    - Variables:                camelCase
    - Constants:                UPPER_SNAKE_CASE
    - Enum Values:              UPPER_SNAKE_CASE
    - Functions:                camelCase()
    - Classes:                   UpperCamelCase


3. Arduino Naming Conventions
    - Variables:                camelCase
    - Constants:                UPPER_SNAKE_CASE
    - Functions:                camelCase()
